== Description ==

Pages Autolink is a SEO plugin which automatically turns page names, when used in posts and pages, in links to the corresponding page URLs. Keeps the original words case in text. It's particulary useful for bloggers who build a glossary using pages, for each time a word of the glossary is used in the site it is automatically converted in a link to its explanation.

== Installation ==

1. Upload the pages-autolink folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You're done.

== Frequently Asked Questions ==

= How many instances of the page names are turned into links? =

Each and all instance.
